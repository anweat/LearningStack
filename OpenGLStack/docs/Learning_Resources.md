# OpenGL 学习资源与实践指南

## 📖 推荐书籍

### 入门级

1. **《OpenGL 编程指南》（红宝书）**

   - 作者: Dave Shreiner, Graham Sellers
   - 适合: 初学者系统学习
   - 特点: 官方指南，覆盖全面

2. **《OpenGL 超级宝典》（蓝宝书）**
   - 作者: Richard S. Wright Jr.
   - 适合: 实践导向学习
   - 特点: 示例丰富，易于理解

### 进阶级

3. **《Real-Time Rendering》（第 4 版）**

   - 作者: Tomas Akenine-Möller 等
   - 适合: 深入理解图形学理论
   - 特点: 实时渲染圣经

4. **《Physically Based Rendering》**

   - 作者: Matt Pharr 等
   - 适合: PBR 深度学习
   - 特点: 物理基础渲染权威

5. **《Game Engine Architecture》**
   - 作者: Jason Gregory
   - 适合: 引擎架构设计
   - 特点: 工业级经验总结

## 💻 实践项目建议

### 阶段 1-2: 基础项目

1. **简单几何体渲染器**

   - 目标: 绘制立方体、球体、圆柱
   - 技术: VAO/VBO、基础着色器
   - 时间: 1 周

2. **2D 精灵编辑器**

   - 目标: 加载图片，自由移动缩放旋转
   - 技术: 纹理、变换矩阵
   - 时间: 1 周

3. **简单 FPS 摄像机**
   - 目标: WASD 移动，鼠标视角
   - 技术: 摄像机系统、输入处理
   - 时间: 3 天

### 阶段 3-4: 中级项目

4. **光照场景演示**

   - 目标: 多光源、材质系统
   - 技术: Phong 光照、多光源
   - 时间: 2 周

5. **模型查看器**

   - 目标: 加载并展示 3D 模型
   - 技术: Assimp、材质系统
   - 时间: 1 周

6. **后处理效果集合**

   - 目标: 实现 10+种后处理
   - 技术: 帧缓冲、屏幕空间效果
   - 时间: 2 周

7. **PBR 材质球展示**
   - 目标: 不同金属度/粗糙度材质
   - 技术: PBR、IBL
   - 时间: 2 周

### 阶段 5-6: 高级项目

8. **小型游戏引擎**

   - 目标: 可加载场景、编辑实体
   - 技术: ECS、资源管理、渲染队列
   - 时间: 1-2 月

9. **延迟渲染引擎**

   - 目标: 支持大量动态光源
   - 技术: G-Buffer、延迟渲染
   - 时间: 2-3 周

10. **阴影系统演示**
    - 目标: CSM、软阴影
    - 技术: Shadow Mapping、PCF/PCSS
    - 时间: 2 周

## 🐛 常见问题速查

### 黑屏问题

```cpp
// 检查清单:
1. glClear()是否调用?
2. 着色器是否编译成功?
3. VAO是否正确绑定?
4. 顶点数据是否在NDC范围内?
5. 深度测试是否正确设置?
```

### 性能问题

```cpp
// 优化检查:
1. 是否有过多的glUniform调用?
2. 纹理是否绑定过于频繁?
3. 是否需要批处理?
4. 顶点数据是否冗余?
5. 着色器是否有优化空间?
```

### 渲染错误

```cpp
// 调试步骤:
1. 使用RenderDoc捕获帧
2. 检查OpenGL错误码
3. 验证着色器输入输出
4. 检查矩阵变换
5. 确认深度/模板测试设置
```
